¿Que es HTML?

Lenguaje de marcado de hipertexto, es el codigo que se utiliza para estructurar y desplegar una pagina web y sus contenidos. Por ejemplo sus contenidos podrian ser parrafos una lista con vuñetas o imagenes o tablas de datos.


Historia del HTML

La historia de HTML comienza en la década de 1980, cuando Tim Berners-Lee, un científico de la computación en el CERN (Organización Europea para la Investigación Nuclear), estaba buscando una forma de compartir y distribuir información entre los investigadores. Berners-Lee desarrolló un lenguaje de marcado simple llamado "HTML" (HyperText Markup Language) como parte de su proyecto para crear la World Wide Web, que podria llamarse Red Extensa Por Todo El Mundo. Esta estrechamente ligada al desarrollo de la World Wide Web.

Como Surgio El HTML

HTML (Hypertext Markup Language) surgió en la década de 1990 como una forma de estructurar ypresentar contenido en la World Wide Web. Fue creado por Tim Berners-Lee, un físico británico,mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear en Suiza. Berners-Lee quería encontrar una forma de compartir documentos científicos de manera más eficiente entre sus colegas en diferentes ubicaciones geográficas.

Aspectos Importantes

1-Estructura Básica: Todo documento HTML comienza con la declaración "!DOCTYPE html"
2-Elementos: Los elementos HTML están formados por etiquetas que rodean el contenido y proporcionan significado a ese contenido. Por ejemplo, "p" para párrafos, "h1" a "h6" para encabezados, "a" para enlaces, "img" para imágenes, etc.
3-Atributos: Las etiquetas pueden tener atributos que proporcionan información adicional sobre el elemento. Por ejemplo, el atributo href en un elemento "a" especifica la URL a la que enlaza el enlace.
4-Anidamiento: Los elementos HTML pueden anidarse dentro de otros elementos. Por ejemplo, un párrafo "p" puede contener un enlace "a", y este enlace puede contener una imagen "img", y así sucesivamente.
